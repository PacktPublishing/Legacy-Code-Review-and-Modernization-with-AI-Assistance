# Review Code Smells

Review the selected code for maintainability and technical debt.

Focus on:

- long methods
- mixed responsibilities
- duplication
- unclear naming
- hidden side effects
- hard-coded dependencies
- poor testability
- inconsistent error handling
- risky data access behavior

Return:

1. Summary
2. Findings with evidence
3. Risk level
4. Suggested fix
5. Tests required
6. Whether to fix now or defer
