# Review Modernization Plan

Review the selected modernization recommendation or generated plan.

Focus on:

- current state
- proposed target state
- affected projects and packages
- breaking change risk
- behavior compatibility
- security implications
- cloud readiness implications
- test requirements
- rollback strategy

Do not approve broad modernization changes without evidence.

Return:

1. Recommendation summary
2. Benefits
3. Risks
4. Files likely affected
5. Required tests
6. Validation commands
7. Accept, defer, or reject decision
